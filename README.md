# Sistema-Git-GitHub-
Sistema de control de versiones: Git y GitHub

Repositorio académico sobre el tema **Sistema de control de versiones Git y GitHub**.
## Descripción
Este repositorio contiene el desarrollo completo del tema **“Sistema de control de versiones Git y GitHub”**, elaborado con fines académicos.  
Aquí se organiza el material principal del trabajo, incluyendo el informe, la presentación, el cuestionario, la documentación de uso de GitHub, los recursos consultados y las evidencias de la demostración práctica.
El propósito de este repositorio es mostrar de manera clara, ordenada y estructurada el estudio y aplicación de **Git** y **GitHub** como herramientas fundamentales para el control de versiones y la gestión de proyectos digitales.

## Objetivo general
Comprender y presentar el funcionamiento de **Git** y **GitHub** como herramientas de control de versiones, destacando su importancia en la organización, seguimiento y almacenamiento de proyectos.

## Objetivos específicos
- Identificar el concepto de sistema de control de versiones.
- Explicar qué es **Git** y qué es **GitHub**.
- Diferenciar las funciones de Git y GitHub.
- Crear y organizar un repositorio académico.
- Documentar el uso de la herramienta GitHub.
- Presentar evidencias de una demostración práctica.
- Reunir todos los archivos del proyecto en una estructura ordenada.
